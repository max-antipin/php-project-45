# Проект "Игры разума"

Пять консольных игр: "Проверка на чётность", "Калькулятор", "Наибольший общий делитель (НОД)", "Арифметическая прогрессия" и "Простое ли число?".

## Требования
- PHP >= 8.4
- Make
- Git
- Composer

### Установка
```shell
make install
```

### Запуск игр
```shell
make brain-even
# Проверка на чётность

make brain-calc
# Калькулятор

make brain-gcd
# Наибольший общий делитель (НОД)

make brain-progression
# Арифметическая прогрессия

make brain-prime
# Простое ли число?
```
